# Ejercicio Técnico Back End

# API REST para proveer información de los artículos registrados

## Operaciones:
  * Alta
  * Baja
  * Consulta
  * Cambio

## Tecnologías utilizadas y herramientas
  * Java 17
  * Spring Boot 3
  * Postman
  * Hibernate

## Base de datos 
  * PostGreSQL
